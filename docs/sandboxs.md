# Online code playgrounds

- [JS Fiddle](https://jsfiddle.net/) - test your JavaScript, CSS, HTML or CoffeeScript online with JSFiddle code editor;
- [Playcode](https://playcode.io/) - the fastest online code sandbox for quickly and easily make frontend experiments;
- [Code Sandbox](https://codesandbox.io/) - instant IDE and prototyping tool for rapid web development;
- [CodePen](https://codepen.io/) - social development environment for front-end designers and developers;
- [JS Bin](http://jsbin.com/?html,output) - live pastebin for HTML, CSS & JavaScript and etc;
- [.NET Fiddle](https://dotnetfiddle.net/) - test your C# code online with .NET Fiddle code editor;
- [Online C# IDE](https://repl.it/languages/csharp) - online C# IDE;
- [Ideone](https://ideone.com/) - online compiler and debugging tool;
